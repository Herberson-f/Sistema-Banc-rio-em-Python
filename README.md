# Sistema Bancário em Python

Projeto desenvolvido para praticar lógica de programação e manipulação de dados em Python.  
O sistema simula operações bancárias básicas, permitindo que o usuário:

- **Cadastrar conta**
- **Realizar login via CPF**
- **Depositar valores**
- **Sacar valores com validação de saldo**
- **Consultar extrato e saldo atual**

O programa utiliza **listas e dicionários** para armazenar usuários, além de aplicar controle de fluxo, tratamento de erros (`try/except`) e funções separadas para cada operação (saque, depósito, extrato, cadastro e menu principal).

Este projeto demonstra:

- Estruturação de um sistema completo no terminal  
- Manipulação de dados em memória  
- Uso de funções para organizar o código  
- Tratamento de entradas inválidas  
- Construção de um menu interativo
